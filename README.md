## Stack Used

1. Reactjs
2. Nextjs boilerplate for Server side render

## Run Project Locally

First, run the development server:

Install it and run:

```bash
npm install
npm run dev
# or
yarn
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
