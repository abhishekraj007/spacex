## Stack Used

1. Reactjs
2. Nextjs boilerplate for Server side rendering
3. SCSS for styling

## Run Project Locally

Clone the repo

```bash
git clone https://github.com/abhishekraj007/spacex.git
cd spacex
```

Install and run:

```bash
yarn
yarn dev
#or
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
